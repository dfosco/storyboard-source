# Pause Embed Refreshes

## Goals

1. Canvas embeds (prototypes, stories) stop refreshing their iframes when "Pause Embed Refreshes" is toggled on
2. Paused embeds resume only when the user interacts with them (clicks to enter interactive mode)
3. Paused embeds show a yellow warning indicator in their header so the user knows they're frozen
4. The toggle is accessible via command palette only, and only shows on canvas routes

## Approach

Use a global DOM class (`storyboard-embeds-paused`) on `document.documentElement` — same pattern as `storyboard-chrome-hidden` for hiding toolbars. Embed widgets check this class and freeze their iframe `src` when paused (unless the user is interacting with that specific widget).

## Files to Change

- `packages/core/src/tools/handlers/pauseEmbeds.js` — **CREATE** — tool handler (toggle class)
- `packages/core/src/tools/registry.js` — **MODIFY** — register the handler
- `packages/core/commandpalette.config.json` — **MODIFY** — add pause-embeds to canvas tools section
- `packages/react/src/canvas/widgets/StoryWidget.jsx` — **MODIFY** — freeze iframe src when paused
- `packages/react/src/canvas/widgets/StoryWidget.module.css` — **MODIFY** — paused header style
- `packages/react/src/canvas/widgets/PrototypeEmbed.jsx` — **MODIFY** — freeze iframe src when paused
- `packages/react/src/canvas/widgets/PrototypeEmbed.module.css` — **MODIFY** — paused header style

## Steps

1. Create `pauseEmbeds.js` handler — toggles `storyboard-embeds-paused` on `document.documentElement`, label flips between "Pause embed refreshes" / "Resume embed refreshes"
2. Register in `registry.js`
3. Add to command palette config (canvas-only via `hideInCommandPalette` regex)
4. In StoryWidget: use a `useSyncExternalStore` or `useState` + MutationObserver to track the paused class. When paused and not interactive, freeze `iframeSrc` to its last value. Add yellow header variant.
5. In PrototypeEmbed: same pattern — freeze iframe src when paused and not interactive. Add yellow header variant.

## Edge Cases & Risks

- Must not break the interactive overlay — user should still be able to click to interact even when paused
- The frozen src should be the LAST valid src, not null — so the iframe shows stale content, not a blank
- When resuming (toggle off), all embeds should update to current src immediately
