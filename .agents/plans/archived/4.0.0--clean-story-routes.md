# Plan: Clean Story Routes

## Goals

1. Story components render with zero margins, padding, borders, or layout constraints at their routes
2. Iframe embeds render just the bare component — no gallery chrome whatsoever

## Problem Statement

StoryPage currently wraps story components in a gallery layout with `max-width: 960px`, margins, padding, headers, badges, bordered content areas, and export labels. When stories are embedded via iframes from canvas StoryWidget, this chrome leaks into the iframe. The routes should render *only* the component itself.

## Approach

Strip all layout chrome from StoryPage. Both single-export and gallery rendering modes should output bare components with no wrapping containers, headers, or borders. Error and loading states keep minimal styling.

## Files to Change

1. `packages/react/src/story/StoryPage.jsx` — remove headers, page containers, section wrappers; render components bare
2. `packages/react/src/story/StoryPage.module.css` — remove `.page`, `.header`, `.title`, `.exportBadge`, `.count`, `.storySection`, `.exportName`, `.storyContent` classes; keep only `.loading` and `.error`

## Steps

1. **Edit StoryPage.jsx**:
   - Single export mode (embed): already clean — keep as-is
   - Single export mode (non-embed): render just `<Component />` (remove page/header/section wrappers)
   - Gallery mode: render each `<Component />` directly (remove page/header/exportName/storyContent wrappers)
   - Error/loading states: keep minimal wrapper for positioning

2. **Edit StoryPage.module.css**:
   - Remove all gallery layout classes (`.page`, `.header`, `.title`, `.exportBadge`, `.count`, `.storySection`, `.exportName`, `.storyContent`)
   - Keep `.loading` and `.error` for functional states

3. **Build + lint + test** to verify nothing breaks

## Edge Cases & Risks

- Gallery mode with multiple exports: components will stack with no visual separation (intentional per user request)
- Error/loading states still need minimal styling to be readable
- The embed mode path is already clean and unchanged
