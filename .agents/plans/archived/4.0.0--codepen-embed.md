# CodePen Embed Widget

## Goals

1. Users can paste a CodePen URL onto a canvas and get an interactive CodePen embed widget (same UX as Figma embeds)
2. The embed shows a title bar with CodePen branding, click-to-interact overlay, expand/collapse modal, and external link

## Approach

Mirror the FigmaEmbed pattern exactly — same lifecycle (click-to-interact, iframe keep-alive, expand modal, external link), same paste-rule mechanism, same widget config structure. CodePen provides an embed URL format: `https://codepen.io/{user}/embed/{penId}`.

## Files to create
- `packages/react/src/canvas/widgets/CodePenEmbed.jsx` — widget component
- `packages/react/src/canvas/widgets/CodePenEmbed.module.css` — styles
- `packages/react/src/canvas/widgets/codepenUrl.js` — URL parser/validator

## Files to modify
- `packages/core/widgets.config.json` — add `codepen-embed` type
- `packages/core/paste.config.json` — add CodePen paste rule
- `packages/react/src/canvas/widgets/index.js` — register component
- `packages/react/src/canvas/widgets/widgetConfig.js` — exclude from menu (paste-only)

## Steps
1. Create `codepenUrl.js` — `isCodePenUrl()`, `toCodePenEmbedUrl()`, `getCodePenTitle()`
2. Create `CodePenEmbed.jsx` copying FigmaEmbed structure
3. Create `CodePenEmbed.module.css` copying FigmaEmbed styles
4. Register in widgets.config.json, paste.config.json, index.js, widgetConfig.js
5. Test with paste rule matching
